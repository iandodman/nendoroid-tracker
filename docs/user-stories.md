# User Stories

## Authentication

* As a collector, I want to sign in with Google so I can access my collection from any device.

## Catalog

* As a collector, I want to browse the official Nendoroid catalog.
* As a collector, I want to search by number, name or series so I can quickly find a Nendoroid.
* As a collector, I want to view detailed information and images for a Nendoroid.

## Collection

* As a collector, I want to add a Nendoroid to my collection.
* As a collector, I want to register multiple copies of the same Nendoroid.
* As a collector, I want to see whether a Nendoroid is already in my collection while browsing the catalog.
* As a collector, I want to view all the Nendoroids I own.

## Wishlist

* As a collector, I want to add a Nendoroid to my wishlist.
* As a collector, I want to know whether a Nendoroid is already in my wishlist while browsing the catalog.

## Home

* As a collector, I want a homepage that immediately lets me search the catalog.
* As a collector, I want quick access to my collection and wishlist.
* As a collector, I want to see the latest releases or upcoming reservations from the home screen.

## Future

* As a collector, I want to receive updates about new releases.
* As a collector, I want to share my collection with other collectors.
* As a collector, I want to view statistics about my collection.
